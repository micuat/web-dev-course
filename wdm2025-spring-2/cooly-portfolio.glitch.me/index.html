üïí Web based Clock

Interactive web-based clock built with HTML, CSS, and JavaScript. It displays real-time updates and supports both light and dark themes with smooth transitions. Users can scale the clock using the mouse wheel, flip it using arrow keys, and toggle fullscreen mode with a single key. Additional features include customizable background colors via RGB picker, font randomization on date click, and responsive layout adjustments on window resize. The project focuses on dynamic styling, user interaction, and adaptive UI design while also going for a pixelated retro look with glowing elements.

üßÆ Math / Physics Calculator

This is a browser-based math and physics calculator originally built in TypeScript, using MathLive for LaTeX input/rendering and the CortexJS Compute Engine for symbolic computation. The interface includes a virtual keypad and dynamic UI updates. Compiled to JavaScript for browser use, the project demonstrates real-time math evaluation, interactive design, and clean TypeScript structuring.
