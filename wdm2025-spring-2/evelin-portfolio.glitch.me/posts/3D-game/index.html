Unfinished 3D game

This is an early experiment with Unity, where I'm learning the basics of building a 3D game. The focus so far has been on testing character movement, adding objects to the environment, and working with simple animations.

The project isn’t in a playable state yet, but it’s been useful for understanding how game logic, physics, and assets come together.

I’m using Unity Hub and gradually learning how to work with its tools and systems through small tests and adjustments.
