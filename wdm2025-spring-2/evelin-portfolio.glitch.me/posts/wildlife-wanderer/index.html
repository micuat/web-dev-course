Wildlife Wanderer

Wildlife Wanderer was a web app for discovering hiking trails in Estonia. It included trail descriptions, filters by difficulty and length, user registration, and comment functionality.

I contributed to the front end, working on the layout, visuals, and features related to user interaction, like logging in, registering, and displaying user comments.

This was a group project made during a university course. The live version is no longer available, but the source code can still be found in GitHub.
