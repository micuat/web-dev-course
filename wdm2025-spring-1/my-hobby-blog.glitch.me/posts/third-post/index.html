How I draw a cat on Ibis

Here you can see, how I use Ibis X.

Tutorial, how to use Ibis


First, I'm creating a sketch.







After that, I create a new layer and change my brush.







For more accurate lines I use "stabilizer" and set it to 3.







After I finish the line, I remove the sketch visibility. I won't need it anymore.






Now I'm choosing "layer selection" and tool named "bucket".






After that I'm filling background and on top clicking on "Reverse Selection Area".







Now the cat itself is selected.






The next step: I'm creating a new folder and there the new layer too. I named it "base". Removing the visibility of "line" and filling the selected area with white.







Afterwards I return the line visibility and removing the selection. And in folder creating a new layer with "clipping". Just select below arrow.






Now I can safely paint my cat on this layer without fear that I might touch the edges of the lines.







When I finished coloring, wanted to add some "Shadows". For that creating a new layer with "clipping" and removing visibility of colored layer. And with that I change my layer from "Normal" mode to "Multiply".







Here you can see,why for me it was smarter toput shadows just on white base - this makes it clearer where to put them.







After that I changed visibility of this layerto 44%.







Now I want to add some lights with creating a new layer (son't forget to clip) and chosing mode "Lighten".







When I finished with lights, want to dosome simpple background and for that I choose tool "brush". On top I choose "Drawing tool" ellipse.







And in the end i made a border.







Here you can see the result of the drawn cat in Ibis. For convenience, I used my tablet (it is not a graphic one, on Android). But the interface and brushes will be the same on the phone. (They will only differ in location). This tutorial showed more how you can use just Ibis.
