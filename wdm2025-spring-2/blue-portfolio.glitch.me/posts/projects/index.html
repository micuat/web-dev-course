Typer game

An interactive web game

This is a typer game, where you need to type the word on the screen on your keyboard as fast as possible and make no mistakes. It is made with HTML, CSS, JavaScript and PHP to store data. It has many features - a leaderboard, new game, asking for the players name, and share your score on Twitter aka X. It also plays sounds during the game and throws confetti when the game is finished.
